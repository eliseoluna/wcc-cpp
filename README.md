# C++ Course Projects

This repository contains a collection of C++ console applications built in Visual Studio for CIS 251. The strongest portfolio-ready projects in this set are listed below.

## Featured Projects

### Project 9: Easter Date Calculator
- Calculates the date of Easter Sunday for the Western Church for years 1583 through 4099
- Demonstrates algorithmic problem solving, loops, input validation, and helper functions
- Files: `Project9/Source.cpp`, `Project9/Project9.sln`, `Project9/Project9.vcxproj`

### Project 10: Membership Rewards Calculator
- Calculates reward points based on membership tier and monthly spending
- Demonstrates function decomposition, conditional logic, and formatted output
- Files: `Project10/Source.cpp`, `Project10/Project10.sln`, `Project10/Project10.vcxproj`

### Project 8: Multiplication Table Generator
- Builds a user-defined multiplication table with range validation and repeated execution
- Demonstrates nested loops, input validation, and table formatting
- Files: `Project8/Source.cpp`, `Project8/Project8.sln`, `Project8/Project8.vcxproj`
