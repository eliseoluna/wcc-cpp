# Project 10: Membership Rewards Calculator

## Overview

This C++ console program calculates customer reward points based on membership tier and monthly purchase amount. The program supports standard, plus, and premium memberships, each with its own reward rules.

## Concepts Demonstrated

- Function decomposition
- Conditional logic
- User input validation
- Formatted output

## Files

- `Source.cpp`: main application logic
- `Project10.sln`: Visual Studio solution
- `Project10.vcxproj`: Visual Studio project file

## How To Run

1. Open `Project10.sln` in Visual Studio.
2. Build and run the project.
3. Choose a membership type and enter a monthly purchase amount.
