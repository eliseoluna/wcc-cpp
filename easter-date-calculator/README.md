# Project 9: Easter Date Calculator

## Overview

This C++ console program calculates the date of Easter Sunday for the Western Church for any valid year from 1583 to 4099. It uses a well-known arithmetic algorithm and formats the result with the correct day suffix such as `st`, `nd`, `rd`, or `th`.

## Concepts Demonstrated

- Algorithm implementation
- Input validation
- Loop-based user interaction
- Helper functions
- String formatting

## Files

- `Source.cpp`: main application logic
- `Project9.sln`: Visual Studio solution
- `Project9.vcxproj`: Visual Studio project file

## How To Run

1. Open `Project9.sln` in Visual Studio.
2. Build and run the project.
3. Enter a year between 1583 and 4099.
4. Enter `0` or a negative number to exit.
