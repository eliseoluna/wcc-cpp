# Project 8: Multiplication Table Generator

## Overview

This C++ console program generates a multiplication table based on a user-selected numeric range. The program validates the starting and ending values, prints a formatted matrix, and allows the user to generate another table without restarting the program.

## Concepts Demonstrated

- Input validation with loops
- Nested `for` loops
- Repeated program flow with a sentinel value
- Formatted console output

## Files

- `Source.cpp`: main application logic
- `Project8.sln`: Visual Studio solution
- `Project8.vcxproj`: Visual Studio project file

## How To Run

1. Open `Project8.sln` in Visual Studio.
2. Build and run the project.
3. Enter a starting and ending value in the allowed range when prompted.
